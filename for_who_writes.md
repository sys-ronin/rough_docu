## How It Works

You see a list of your notebooks. Each notebook has a number: `[1]`, `[2]`, `[3]`.

Below the list, you see commands: `[C]reate  [V]iew  [S]earch  [D]elete  [L]ock  [M]anage  [Q]uit`

The letters tell you what you can do. The numbers tell you which item to do it to.

To open a notebook, press `v` then the notebook's number. `v1` opens notebook 1. `v2` opens notebook 2.

Inside a notebook, your notes are listed with numbers. To view a note, press `v` then the note's number. `v1` views note 1. `v2` views note 2.

Inside a note, you see `[E]dit`, `[V]iew`, `[T]imeline`, `[R]ename`, `[B]ack`, `[Q]uit`. Press `e` to edit. Press `v` to view read-only. Press `t` to see every version ever written.

**You don't memorize. You see. You press.**

## What You Get

### Your Words Never Disappear

Delete a note. It is not gone. It is in history. Type `deleted*` in search. Every deleted note appears. Press `v1` to view it. Press `r` to restore it.

Rename a note. The history follows the content, not the name. Type `renamed*` in search. Every note that ever changed names appears. You see what it was called before. You see what it is called now.

Every version of every note exists. Forever. You can go back to any version. You can restore any deleted item.

### Your Words Are Private

Every notebook is encrypted with AES-256-GCM. The same standard used by banks and governments.

You choose a password. You write down a recovery phrase. The phrase is shown once. It is never stored anywhere. Only you have it.

No one can read your notes without your password and phrase. Not the developer. Not the company. Not any company. No one.

### Your Words Follow You

A notebook is a single folder. Copy it to a USB drive. Copy it to another computer. Copy it to the cloud. The folder contains everything.

On a new machine, enter your recovery phrase once. Then use your password daily. The notebook works everywhere.

### Your Words Sync for Free

Every notebook is a Git repository. Push to GitHub. Push to GitLab. Push to any Git server. The files are encrypted before they leave your computer. Public repositories are safe. Private repositories are private.

No subscription. No vendor lock-in. No cloud to trust.

### You Can Write Anything

Not just plain text. Code files with syntax highlighting. Configuration files. Markdown. LaTeX. JSON. YAML. Any text-based file.

Create a file with `.py` extension. Edit with full syntax highlighting. Git tracks every change. Encryption protects every version.

One tool. All your writing.

### You Can Find Anything

Search is not just keywords. You can search by what happened:

- `created*` — notes you created
- `deleted*` — notes you deleted
- `renamed*` — notes that changed names
- `updated*` — notes you edited

You can search by type:

- `note*` — only text notes
- `file*` — only code and config files

You can search by time:

- `today*` — what you wrote today
- `yesterday*` — what you wrote yesterday
- `thisweek*` — what you wrote this week
- `lastweek*` — what you wrote last week

You can search by location:

- `in* notebook_name` — only notes in a specific notebook

Combine them: `deleted* file* yesterday* in* work`

One search. Everything you have ever written. Deleted? Still there. Renamed? Still there.
